AI Self-Grade Assistant — load in Chrome

1. Open chrome://extensions
2. Turn on "Developer mode"
3. Click "Load unpacked"
4. Select this folder: chrome-extension (the one containing manifest.json)

Supported sites (content script runs on):
- chatgpt.com / chat.openai.com
- claude.ai
- gemini.google.com
- perplexity.ai

How it works:
- The extension cannot see your files or videos. It only inserts prompts so the model states whether it actually accessed attachments and flags ungrounded or fabricated details.
- Panel actions: "Append full epistemic + confidence block" (before your question), "Insert media / file access check" (short block focused on video/audio/PDF honesty), "Insert self-grade + fabrication audit" (after a reply you distrust).
- Options page: edit all three snippets or reset to defaults (puzzle icon → AI Self-Grade Assistant → options).

Note: Some chat UIs use shadow DOM for the input; if the panel says it cannot find the input, click inside the chat box first or the site may have changed its markup.
