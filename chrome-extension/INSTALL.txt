Caliber AI — ai self grade assistant (Chrome extension, load locally)

Name in chrome://extensions: **Caliber AI - ai self grade assistant** (unpacked from this repo).

Not the same as the Web Store “Caliber AI” overlay (v1.0.0, different ID). This build pastes self-grade prompts; it does not draw confidence overlays on messages.

What to download
- Repo ZIP from GitHub → unzip → use the **chrome-extension** folder (contains manifest.json, content/, shared/).

Load in Chrome
1. chrome://extensions → Developer mode ON
2. Load unpacked → select the **chrome-extension** folder
3. After updates: **Reload** the extension, then refresh the chat tab

Panel disappears after reload?
- Chat sites often wipe the page DOM. This version re-attaches the panel when that happens. Use the latest **chrome-extension** from GitHub and reload the extension.

Supported sites
- chatgpt.com / chat.openai.com, claude.ai, gemini.google.com, perplexity.ai

Usage
- Bottom-right **Caliber AI** panel: buttons paste into the chat input; you click **Send**.
- If paste fails, click inside the chat box first.
