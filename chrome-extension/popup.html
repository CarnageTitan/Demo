Caliber — Chat Prompts

Wrong extension? If this popup does not match what you see (e.g. “Caliber AI”, shield icon, “Confidence Score Overlay”, v1.0.0), you installed a different app from the Chrome Web Store. Remove it and use Load unpacked with the chrome-extension folder from this project.

This extension adds a dark Caliber panel at the bottom-right of the chat page with buttons that paste text into the input. It does not add a floating score widget on messages.

It only runs on supported AI chat URLs—not on Google, YouTube, or blank tabs.

1. Open one of these (click to open a new tab):

ChatGPT Claude Gemini Perplexity

2. After the chat loads, look for a dark Caliber panel at the bottom-right. Click a button there to paste prompts into the chat box, then send your message.

3. If you just installed Caliber, refresh the chat tab (F5 or reload).
