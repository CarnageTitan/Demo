Caliber AI — self-grade

Store vs this build: The Web Store app “Caliber AI” (confidence overlay, v1.0.0) is different. This one is Load unpacked from GitHub and pastes prompts—no score overlay on messages.

A dark Caliber AI panel stays at the bottom-right on chat pages. Buttons paste text into the input; you press Send.

It only runs on supported AI chat URLs—not on Google, YouTube, or blank tabs.

1. Open one of these (click to open a new tab):

ChatGPT Claude Gemini Perplexity

2. If the panel vanishes after reload, update to the latest GitHub zip and click Reload on chrome://extensions—this build re-attaches after the site redraws.

3. After install, refresh the chat tab (F5).
